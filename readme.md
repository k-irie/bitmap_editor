# ビットマップ エディタ

## 概要

ビットマップ画像編集ソフト

## 機能

以下の機能を実装している

- キャンバスサイズの変更
- カラーモード(カラー、モノクローム)
- カラーパレット
    - パレットの色変更(カラーモード時)
- ズーム(1倍～50倍まで:既定値10倍)
- ドローツール
    - 鉛筆
    - 直線
    - 塗り潰し
- プレビュー
- カーソル位置
- ファイル出力
    - PNG
    - BMP
    - ICO
    - C言語配列変数
        - 2進数
        - 8進数
        - 16進数
- 初期値外部ファイル化
